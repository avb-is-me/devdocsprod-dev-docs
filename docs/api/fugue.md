## fugue.constants


### fugue.constants.register_global_conf(conf, on_dup=0)
Register global Fugue configs that can be picked up by any
Fugue execution engines as the base configs.


* **Parameters**


* **conf** (*Dict**[**str**, **Any**]*) -- the config dictionary


* **on_dup** (*int*) -- see [`triad.collections.dict.ParamDict.update()`](https://triad.readthedocs.io/en/latest/api/triad.collections.html#triad.collections.dict.ParamDict.update)
, defaults to `ParamDict.OVERWRITE`



* **Return type**

None


**NOTE**: When using `ParamDict.THROW` or `on_dup`, it's transactional.
If any key in `conf` is already in global config and the value
is different from the new value, then ValueError will be thrown.
